## Day plan

| Time | Session |
|---|---|
| 9:00–10:30 | **Foundations refresher.** Quick review of canonical 2×2 DiD; potential outcomes; parallel trends. Level-set across mixed methods exposure. |
| 10:30–10:45 | Coffee break |
| 10:45 AM–12:15 PM | **Goodman-Bacon decomposition + the TWFE problem.** Why TWFE is biased under staggered adoption + heterogeneous effects. Negative weights. Why the field changed. |
| 12:15–1:15 PM | Lunch |
| 1:15–2:45 PM | **Modern estimators.** Callaway-Sant'Anna; Sun-Abraham; Borusyak-Jaravel-Spiess. Aggregation schemes. Event-study specifications. Hands-on examples. |
| 2:45–3:00 PM | Coffee break |
| 3:00–4:00 PM | **Sensitivity & frontier.** Roth-Sant'Anna and Rambachan-Roth sensitivity bounds; continuous-treatment DiD; synthetic DiD. Choosing methods in practice. Q&A. |

This is a streamlined version of the Pisa Day 1+2 material, trimmed to fit a single day.

- Faculty audience expects a recent-literature framing — emphasize what's changed since 2020
